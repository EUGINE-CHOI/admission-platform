## 4. 기능 요구사항 (업데이트)

### 4.1 계정 시스템 (유지)
- 가족 계정 구조 강화  
- 학생 입력 → 부모 승인 흐름 유지
- **역할별 온보딩 튜토리얼 (신규)**

### 4.2 학생 데이터 입력 (유지)
- 성적/활동  
- 부정확한 입력 방지: 점수 범위 검증 (0~100)
- **성적 목표 설정 기능 (신규)**

### 4.3 전형 정보 (개선)
- 크롤링 → 관리자 승인  
- 입시 일정 데이터 자동 수집 + 표준화  
- 학교별 요약 카드 제공
- **학교 간 비교 분석 (신규)**
- **학교 홈페이지 바로가기 (신규)**

### 4.4 진단 엔진 (강화)
- 기존: FIT/CHALLENGE/UNLIKELY  
- 개선:
  - 전형별 전략 추천
  - 커트라인 대비 차이 분석
  - AI 기반 "3개월 준비 플랜"
  - **합격 확률 예측 (신규)**
  - **성적 트렌드 분석 (신규)**
