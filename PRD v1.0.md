# 📘 입시 정보 격차 해소 플랫폼 — PRD v1.0 (MVP 기준)

## 1. 서비스 개요
입시 정보 접근성이 낮고 전문 컨설팅 비용이 부담되는 학생/학부모를 위해  
입시 정보, 학생 데이터 기반 진단, AI 기반 코칭을 제공하는 고입 입시 플랫폼.

## 2. 주요 사용자
- 학생(초6~중3)
- 학부모(부모 1명 → 자녀 1/2명 관리)
- 컨설턴트
- 관리자

## 3. 서비스 범위(MVP)
- 고입 특목고/자사고/일반고 정보 제공  
- 학생 데이터 입력(성적, 활동, 독서 등)  
- AI 기반 학생부/동아리/교과/독서 분석  
- 고입 진단 알고리즘(FIT/CHALLENGE/UNLIKELY)  
- 크롤링 기반 전형 정보 자동화  
- 상담 예약 및 컨설턴트 기능  
- 결제/구독 모델  
- 학부모 대시보드  
- PDF 보고서

---

# 4. 기능 요구사항

## 4.1 계정/역할
- STUDENT / PARENT / CONSULTANT / ADMIN  
- 부모→자녀 연결(family_id 기반)

## 4.2 학생 데이터 입력
- 프로필  
- 성적(지필/수행)  
- 학생부 활동(세특/창체/동아리/독서)  
- 부모 승인(확인)

## 4.3 전형 정보
- 학교별 전형 수집(크롤링)  
- Draft → Admin Approval → Publish  
- 일정·전형·요약 정보 제공

## 4.4 진단 알고리즘
- 입력: 내신 + 활동 + 성향  
- 출력: FIT, CHALLENGE, UNLIKELY  
- 추천 학교 리스트 및 이유

## 4.5 AI 에이전트(4종)
- 학생부 도우미  
- 동아리 추천  
- 교과 조언  
- 독서 추천 및 독후 활동

## 4.6 결제/구독
- 월 구독 모델  
- 특목고/자사고/일반고 등급별 가격  
- 가족 할인(두 자녀 사용 시 10%)

## 4.7 컨설턴트 기능
- 상담 예약  
- 메시지  
- 상담 리포트 제공

## 4.8 학부모 대시보드
- 학생 활동 요약  
- 성적/활동/AI 분석  
- 진단 보고서  
- PDF 생성
