**산출물:**

## WP17.2 — 하단 네비게이션 바

**목표:** 모바일 주요 메뉴 빠른 접근

**산출물:**
- 역할별 주요 메뉴 표시
- 모바일에서만 표시
- 현재 페이지 하이라이트

→ 한 손 조작으로 주요 기능 접근

## WP17.3 — 성능 최적화

**목표:** 전반적인 로딩 속도 향상

**산출물:**
- 이미지 최적화 (WebP/AVIF)
- 지연 로딩 (Lazy Loading)
- 스켈레톤 UI 적용
- API 캐싱 (TTL)

→ 빠른 페이지 로딩 경험

---

## WP17.4 — 모바일 테스트 환경

**목표:** 모바일 기기 테스트 지원

**산출물:**
- 동적 API URL (getApiUrl)
- CORS 확장 (모든 origin 허용)
- 전체 네트워크 바인딩 (0.0.0.0)

→ 같은 네트워크에서 모바일 테스트 가능
