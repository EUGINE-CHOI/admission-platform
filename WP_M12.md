## WP12.1 — 중학교 데이터베이스

**목표:** 전국 중학교 데이터 구축

**산출물:**
- 서울/인천/경기 341개 중학교 데이터
- 중학교 검색 API
- 지역별 필터링

**데이터 규모:**
| 지역 | 학교 수 |
|------|--------|
| 서울 | 120개 |
| 인천 | 85개 |
| 경기 | 136개 |

→ 회원가입 시 재학 중학교 선택 가능

## WP12.2 — 특목고/자사고 확장

**목표:** 목표 학교 선택 범위 확대

**산출물:**
- 전국 특목고 데이터 (외고, 과학고, 국제고)
- 전국 자사고 데이터
- 학교 상세 정보 (정원, 전형, 특징)

**데이터 규모:**
| 유형 | 학교 수 |
|------|--------|
| 외고 | 30개 |
| 과학고 | 20개 |
| 국제고 | 8개 |
| 자사고 | 34개 |

→ 92개 특목고/자사고 선택 가능

## WP12.3 — 입시 경쟁률 히스토리

**목표:** 학교별 경쟁률 추이 제공

**산출물:**
- 2019~2025년 경쟁률 데이터
- 학교별 경쟁률 추이 API
- 전형별 경쟁률 비교

**API:**
| Method | Path | 설명 |
|--------|------|------|
| GET | `/api/schools/:id/history` | 경쟁률 히스토리 |

→ 경쟁률 기반 목표 설정 지원
