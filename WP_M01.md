🧩 M1 — 계정/가족 계정 시스템 (Identity Layer)
WP1.1 — 학생/학부모/컨설턴트 회원가입 & 로그인

이메일/비밀번호 + JWT

→ 사용자가 로그인 가능한 상태

WP1.2 — Role 기반 권한관리 RBAC

Student/Parent/Consultant/Admin 권한

→ API 접근이 Role에 따라 제한됨

WP1.3 — 가족 연결(Parent → Student 초대 코드)

초대 코드 생성/확인

→ 부모가 자녀 계정에 연결 가능

WP1.4 — 학부모가 자녀 프로필 열람

자녀의 기본 정보 표시

→ 부모 UI에서 자녀 정보 확인 가능

WP1.5 — 재학 중학교 및 학년 정보 입력

회원가입 시 학생/학부모가 현재 재학 중인 중학교명과 학년을 입력

입력된 정보는 대시보드에 표시되어 사용자 경험 향상

→ 학생 대시보드에서 "🏫 OO중학교 O학년" 형태로 표시

WP1.6 — 중학교 데이터베이스 및 검색 기능

서울/인천 지역 중학교 데이터베이스 구축 (150개 학교)

회원가입 시 학교명 실시간 검색 (한 글자부터 검색 가능)

학교 선택 시 자동으로 지역, 구/군, 홈페이지 URL 연결

학생 대시보드에서 학교 홈페이지 바로가기 링크 제공

→ 회원가입 UX 향상 및 정확한 학교 데이터 확보

WP1.7 — 동아리 데이터베이스 및 검색/추천 기능

동아리 데이터베이스 구축:
- 45개 일반 동아리 템플릿 (학술, 예술, 체육, 봉사, 진로, 문화)
- 서울/인천 10개 중학교 크롤링 (95개 실제 동아리 데이터)
- 총 140개 동아리 데이터

학생 동아리 페이지 기능:
- 카테고리별 동아리 필터링 (학술/예술/체육/봉사/진로/문화)
- 키워드 검색 기능
- 내 동아리 등록 및 관리
- 동아리 추천 (관심사 기반)

동아리 API:
- GET /api/clubs - 동아리 검색 (keyword, category, schoolId 필터)
- GET /api/clubs/general - 일반 동아리 템플릿 목록
- GET /api/clubs/categories - 카테고리별 통계
- POST /api/clubs/recommend - 관심사 기반 동아리 추천

→ 학생이 적합한 동아리를 찾고 활동을 관리할 수 있음
