## 📋 흐름도 설명

### 1. 진입점 (Entry Points)

| 진입점 | 설명 |
|--------|------|
| Landing/Login | 메인 페이지, 로그인 선택 |
| Student 선택 | 학생 역할로 진입 |
| Parent 선택 | 학부모(보호자) 역할로 진입 |
| Consultant 선택 | 컨설턴트 역할로 진입 |
| **🎮 데모 체험** | 테스트 계정으로 즉시 체험 (신규) |

### 2. Student Flow (학생 흐름)

#### 기존 흐름
```
Onboarding → Dashboard → 데이터 입력 → 진단 → AI 플랜 → 목표 학교 가이드
```

#### 신규 기능 (v1.2)
| 기능 | 설명 |
|------|------|
| 🎯 목표 성적 설정 | 과목별 목표, 진행률 추적 |
| 📈 성적 트렌드 | 과목별 추이, 개선 조언 |
| 🏫 학교 비교 | 목표 학교 상세 비교 |
| 📝 AI 자기소개서 | 초안 생성, AI 첨삭 |
| 🎤 면접 준비 | 예상 질문, 모의 면접 |
| 📰 최신 뉴스 | 특목고 관련 뉴스 |
| 💬 커뮤니티 Q&A | 질문/답변, 좋아요 |
| ✨ 합격생 후기 | 후기 열람, 댓글 |
| 🏆 뱃지 획득 | 활동별/목표 달성 뱃지 |

### 3. Parent Flow (학부모 흐름)

#### 기존 흐름
```
Onboarding → Dashboard → 자녀 활동 검토 → 진단 열람 → 캘린더 → PDF 리포트
```

#### 신규 기능 (v1.2)
| 기능 | 설명 |
|------|------|
| 📈 성적 추이 차트 | 자녀 성적 그래프 |
| 🏫 학교 비교 | 목표 학교 비교 |
| 🎲 합격 시뮬레이션 | 가상 시나리오 테스트 |
| 📰 최신 뉴스 | 입시 관련 뉴스 |
| 💬 커뮤니티 Q&A | 학부모 질문/답변 |
| D-Day 카운트다운 | 대시보드 상단 표시 |

### 4. Consultant Flow (컨설턴트 흐름)

#### 기존 흐름
```
Dashboard → 학생 현황 → 상담 진행 → 리포트 작성
```

#### 신규 기능 (v1.2)
| 기능 | 설명 |
|------|------|
| 📈 성적 트렌드 보기 | 학생별 성적 추이 |
| 📅 상담 예약 관리 | 캘린더 연동 |
| PDF 생성 | 상담 리포트 PDF |

### 5. Admin Flow (관리자 흐름) - 신규 ⭐

```
Admin Dashboard → 학교/전형 관리 → AI 품질 관리 → 사용자 관리 → 커뮤니티 관리
```

| 기능 | 설명 |
|------|------|
| 🏫 학교/전형 관리 | 크롤링 데이터 승인 |
| 🤖 AI 품질 관리 | AI 출력 모니터링 |
| 👥 사용자 관리 | 컨설턴트 승인 |
| 💬 커뮤니티 관리 | 신고 처리, 후기 검증 |

## 🔗 Cross Links (역할 간 연결)

| 연결 | 설명 |
|------|------|
| 학생 → 학부모 | 초대 코드로 가족 연결 |
