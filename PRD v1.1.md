# 4. 기능 요구사항 (업데이트)

## 4.1 계정 시스템 (유지)
- 가족 계정 구조 강화

## 4.2 학생 데이터 입력 (유지)

## 4.3 전형 정보 (개선)

- 학교별 요약 카드 제공

---

## 4.4 진단 엔진 (강화)
- 기존: FIT/CHALLENGE/UNLIKELY  
- 개선:
  - **전형별 전략 추천**
  - **커트라인 대비 차이 분석**
  - **AI 기반 “3개월 준비 플랜”**
