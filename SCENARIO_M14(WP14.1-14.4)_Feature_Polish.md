## 📅 WP14.1 — 상담 예약 캘린더

### Scenario WP14.1-1: 통합 캘린더 표시

**Given:**
- 학생이 로그인되어 있다
- 입시 일정과 상담 예약이 존재한다

**When:**
- 캘린더 페이지 접속

**Then:**
- 월간 캘린더 표시
- 입시 일정 (파란색) + 상담 예약 (녹색) 통합 표시
- 날짜 클릭 시 해당 일정 상세

**선행 Scenario:** WP1.1-3

---

### Scenario WP14.1-2: 캘린더에서 상담 예약

**Given:**
- 캘린더 페이지에 있다
- 컨설턴트가 배정되어 있다

**When:**
- 빈 날짜 클릭 → "상담 예약" 선택

**Then:**
- 상담 예약 모달 표시
- 시간대 선택 가능
- 예약 완료 시 캘린더에 반영

**선행 Scenario:** WP14.1-1

## 📄 WP14.2 — PDF 리포트 생성

### API 명세

| Method | Endpoint | 설명 |
|--------|----------|------|
| GET | `/api/report/student/:id/pdf` | 학생 리포트 PDF |
| GET | `/api/report/diagnosis/:id/pdf` | 진단 리포트 PDF |

---

### Scenario WP14.2-1: 학생 리포트 PDF 다운로드

**Given:**
- 학생 대시보드에 있다
- 성적, 활동 데이터가 입력되어 있다

**When:**
- "PDF 다운로드" 버튼 클릭

**Then:**
- PDF 파일 생성 및 다운로드 시작
- 파일명: `학생리포트_홍길동_20250115.pdf`
- 내용: 성적 요약, 활동 목록, 진단 결과

**선행 Scenario:** WP2.1-1

---

### Scenario WP14.2-2: 진단 리포트 PDF 다운로드

**Given:**
- 진단 결과 페이지에 있다
- 진단이 완료되어 있다

**When:**
- "진단 결과 PDF" 버튼 클릭

**Then:**
- PDF 파일 다운로드
- 내용: 적합 학교 목록, 상세 분석, 추천 사항

**선행 Scenario:** WP4.2-1

## 👨‍👩‍👧 WP14.3 — 보호자 대시보드 상세화

### Scenario WP14.3-1: 성적 추이 차트 표시

**Given:**
- 보호자가 로그인되어 있다
- 연결된 자녀의 성적 데이터가 있다

**When:**
- 보호자 대시보드 접속

**Then:**
- 자녀 성적 추이 차트 표시
- 과목별 성적 변화 그래프

**선행 Scenario:** WP1.3-2

---

### Scenario WP14.3-2: 학생별 상세 페이지

**Given:**
- 보호자 대시보드에 있다
- 여러 자녀가 연결되어 있다

**When:**
- 특정 자녀 카드 클릭

**Then:**
- 해당 자녀 상세 페이지로 이동
- 성적, 활동, 일정, 진단 결과 통합 표시

**선행 Scenario:** WP14.3-1

---

### Scenario WP14.3-3: 활동 요약 카드

**Given:**
- 보호자 대시보드에 있다

**When:**
- 페이지 로드

**Then:**
- 자녀별 활동 요약 카드 표시:
  - 최근 활동 수
  - 완료된 Task 수
  - 다가오는 일정 수

**선행 Scenario:** WP14.3-1

---

## 📝 WP14.4 — 용어 통일

### Scenario WP14.4-1: 학부모 → 보호자 전환

**Given:**
- 시스템 전체에서 "학부모" 용어 사용 중

**When:**
- 용어 통일 작업 완료

**Then:**
- 모든 UI에서 "보호자"로 표시
- 역할 표시: "보호자"
- 메뉴 명칭: "보호자 대시보드"

**선행 Scenario:** 없음

---

### Scenario WP14.4-2: 자녀 → 학생 전환

**Given:**
- 보호자 관점 UI에서 "자녀" 용어 사용 중

**When:**
- 용어 통일 작업 완료

**Then:**
- 보호자 대시보드에서 "학생" 또는 "자녀"로 통일
- 문맥에 맞는 용어 사용

**선행 Scenario:** WP14.4-1
