🧩 M3 — 학교/전형/입시일정 자동 수집 & 관리자 승인 시스템 (Admissions Layer)
🎯 목적

전형 정보·일정을 자동화하여 데이터 수집 비용을 최소화하고 정확도 보장.

📦 산출물

Python + Playwright 크롤러 구조 (또는 NestJS + Puppeteer)

schools / admissions / schedules 테이블 구축

관리자(Admin) 승인 Pipeline

프론트에서 학교별 전형 정보 조회

전형/일정 버전 관리(Versioning 적용)

🔗 의존성

진단 엔진(M4)은 이 전형 데이터를 반드시 사용.

학부모 캘린더(M7)도 이 데이터에 의존.

✅ 추가 기능 (2024.12 업데이트)

- 학교 데이터에 website 필드 포함
- 진단 페이지에서 목표 학교 옆 홈페이지 링크 아이콘 표시
- 클릭 시 새 탭에서 학교 공식 홈페이지 열림

✅ 데이터 확장 (2024.12.11 업데이트)

**전국 중학교 데이터베이스 구축**
- 총 341개 중학교 데이터 (서울 110개, 경기 76개, 인천 40개, 부산 31개, 대구 24개, 광주 20개, 대전 18개, 울산 13개, 세종 9개)
- 학교명, 지역, 구/군, 홈페이지 URL 포함
- 회원가입 시 중학교 검색/선택 기능 연동

**전국 특목고/자사고 데이터 확장**
- 총 92개 고등학교 데이터
- 과학고/영재학교 19개, 외국어고 22개, 국제고 5개, 예술고 7개, 체육고 4개, 자사고 20개
- 학교 유형별 세분화 (SchoolType enum 확장)

**2025학년도 입시 일정 데이터**
- 총 109개 입시 일정 데이터
- 입학설명회, 원서접수, 서류심사, 시험, 면접, 합격자 발표, 등록 일정 포함
- 주요 특목고/자사고 2025학년도 전형 일정 수록

✅ 실제 크롤링 시스템 구현 (2024.12.12 업데이트)

**Puppeteer 기반 실제 학교 홈페이지 크롤링**
- 학교 홈페이지에서 입시 정보 자동 수집
- 입시 페이지 자동 탐색 (URL 패턴 분석)
- 일정 정보 추출 (날짜 패턴 인식)
- 전형 요소 추출 (내신/면접 비율)
- 제출 서류 정보 추출

**관리자 크롤링 관리 UI**
- 학교 유형별 일괄 크롤링 (과학고, 외국어고, 국제고, 자사고)
- 개별 학교 선택 크롤링
- 크롤링 결과 실시간 확인
- 크롤링 히스토리 조회
- 수집 데이터 관리자 승인 워크플로우

**API 엔드포인트**
- POST /api/admin/crawler/real/school/:schoolId (단일 학교 크롤링)
- POST /api/admin/crawler/real/schools (다중 학교 크롤링)
- POST /api/admin/crawler/real/type/:type (유형별 크롤링)
- GET /api/admin/crawler/real/available-schools (크롤링 가능 학교 목록)
- GET /api/admin/crawler/real/crawl-history (크롤링 히스토리)
