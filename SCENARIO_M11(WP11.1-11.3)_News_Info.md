## 🔧 WP11.1 — Google News RSS 크롤링

### API 명세

| Method | Endpoint | 설명 |
|--------|----------|------|
| GET | `/api/news` | 뉴스 목록 조회 |
| GET | `/api/news/:id` | 뉴스 상세 조회 |

---

### Scenario WP11.1-1: 뉴스 목록 조회 성공

**Given:**
- 사용자가 로그인되어 있다
- Google News RSS에서 뉴스가 수집되어 있다

**When:**
- `GET /api/news?keyword=외고`

**Then:**
- 응답:
```json
{
  "success": true,
  "data": [
    {
      "id": "news-1",
      "title": "2025학년도 외고 입시 변화",
      "source": "교육일보",
      "publishedAt": "2025-01-15T10:00:00Z",
      "url": "https://news.google.com/...",
      "snippet": "올해 외고 입시에서..."
    }
  ]
}
```

**선행 Scenario:** WP1.1-3

---

### Scenario WP11.1-2: 키워드별 뉴스 필터링

**Given:**
- 뉴스 목록 페이지에 접속
- 다양한 키워드의 뉴스가 존재

**When:**
- 키워드 탭에서 "과학고" 선택

**Then:**
- 과학고 관련 뉴스만 필터링되어 표시
- 다른 키워드 뉴스는 숨겨짐

**선행 Scenario:** WP11.1-1

---

### Scenario WP11.1-3: 뉴스 캐싱 동작

**Given:**
- 뉴스가 캐싱되어 있다 (TTL: 30분)

**When:**
- 30분 이내에 동일 키워드 뉴스 요청

**Then:**
- 캐시된 뉴스 반환 (API 호출 없음)
- 응답 시간 < 100ms

**선행 Scenario:** WP11.1-1

---

## 📄 WP11.2 — 뉴스 상세 모달

### Scenario WP11.2-1: 뉴스 상세 모달 표시

**Given:**
- 뉴스 목록이 표시되어 있다

**When:**
- 뉴스 항목 클릭

**Then:**
- 뉴스 상세 모달이 표시
- 제목, 출처, 발행일, 본문 스니펫 표시
- 원문 보기 버튼 표시

**선행 Scenario:** WP11.1-1

---

### Scenario WP11.2-2: 뉴스 원문 링크 이동

**Given:**
- 뉴스 상세 모달이 열려 있다

**When:**
- "원문 보기" 버튼 클릭

**Then:**
- 새 탭에서 원본 뉴스 기사 페이지 열림

**선행 Scenario:** WP11.2-1

---

### Scenario WP11.2-3: 뉴스 북마크 저장

**Given:**
- 뉴스 상세 모달이 열려 있다

**When:**
- 북마크 아이콘 클릭

**Then:**
- 뉴스가 북마크에 저장
- "북마크에 저장되었습니다" 토스트 표시

**선행 Scenario:** WP11.2-1

---

### Scenario WP11.2-4: 뉴스 링크 복사

**Given:**
- 뉴스 상세 모달이 열려 있다

**When:**
- "링크 복사" 버튼 클릭

**Then:**
- 뉴스 URL이 클립보드에 복사
- "링크가 복사되었습니다" 토스트 표시

**선행 Scenario:** WP11.2-1

---

## 🎮 WP11.3 — 데모 체험 모드

### Scenario WP11.3-1: 데모 로그인 버튼 클릭

**Given:**
- 랜딩 페이지에 접속

**When:**
- "체험하기" 버튼 클릭

**Then:**
- 테스트 계정으로 자동 로그인
- 학생 대시보드로 리다이렉트
- "체험 중" 뱃지 표시

**선행 Scenario:** 없음

---

### Scenario WP11.3-2: 체험 모드 뱃지 표시

**Given:**
- 데모 계정으로 로그인되어 있다

**When:**
- 대시보드 페이지 표시

**Then:**
- 화면 상단에 "체험 중" 뱃지 표시
- 뱃지에 "데모 계정입니다" 툴팁

**선행 Scenario:** WP11.3-1

---

### Scenario WP11.3-3: 체험 모드 기능 제한

**Given:**
- 데모 계정으로 로그인되어 있다

**When:**
- 결제/구독 페이지 접근 시도

**Then:**
- "체험 모드에서는 이용할 수 없습니다" 메시지 표시
- 실제 결제 진행 불가

**선행 Scenario:** WP11.3-1
