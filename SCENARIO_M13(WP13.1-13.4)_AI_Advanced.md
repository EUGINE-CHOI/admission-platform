## ⚡ WP13.1 — 빠른 조언 (Quick Advice)

### API 명세

| Method | Endpoint | 설명 |
|--------|----------|------|
| POST | `/api/ai/quick-advice` | 빠른 조언 요청 |

---

### Scenario WP13.1-1: 빠른 조언 요청 성공

**Given:**
- 학생이 로그인되어 있다
- AI 멘토 페이지에 있다

**When:**
- 빠른 조언 요청: "내신 3등급인데 외고 갈 수 있어?"

**Then:**
- 2초 이내 응답
- 간결한 조언 제공 (3~5문장)
- 추가 질문 제안

**선행 Scenario:** WP1.1-3

---

### Scenario WP13.1-2: 카테고리별 빠른 조언

**Given:**
- AI 멘토 페이지에 있다

**When:**
- 카테고리 "성적" 선택 후 질문

**Then:**
- 성적 관련 맞춤형 조언 제공
- 관련 데이터 참조하여 답변

**선행 Scenario:** WP13.1-1

---

## 📊 WP13.2 — 종합 분석 (Comprehensive Analysis)

### API 명세

| Method | Endpoint | 설명 |
|--------|----------|------|
| POST | `/api/ai/analysis` | 종합 분석 요청 |

---

### Scenario WP13.2-1: 종합 분석 요청 성공

**Given:**
- 학생 데이터가 입력되어 있다 (성적, 활동)

**When:**
- "종합 분석" 버튼 클릭

**Then:**
- 현황 분석 결과 표시:
  - 강점 영역 (2~3개)
  - 약점 영역 (2~3개)
  - 핵심 개선 조언 (3~5개)
- 응답 시간 < 10초

**선행 Scenario:** WP2.1-1, WP2.2-1

---

### Scenario WP13.2-2: 분석 결과 저장

**Given:**
- 종합 분석 결과가 표시되어 있다

**When:**
- "저장" 버튼 클릭

**Then:**
- 분석 결과가 히스토리에 저장
- "저장되었습니다" 토스트 표시

**선행 Scenario:** WP13.2-1

---

## 🏫 WP13.3 — AI 학교 추천

### API 명세

| Method | Endpoint | 설명 |
|--------|----------|------|
| POST | `/api/ai/school-recommendations` | 학교 추천 요청 |

---

### Scenario WP13.3-1: 학교 추천 요청 성공

**Given:**
- 학생 데이터가 충분히 입력되어 있다

**When:**
- "학교 추천받기" 버튼 클릭

**Then:**
- 맞춤형 학교 리스트 (3~5개) 제공
- 각 학교별 적합도 점수 표시
- 추천 이유 설명

**선행 Scenario:** WP2.1-1, WP2.2-1

---

### Scenario WP13.3-2: 적합도 점수 상세

**Given:**
- 학교 추천 결과가 표시되어 있다

**When:**
- 특정 학교의 "상세" 버튼 클릭

**Then:**
- 적합도 점수 분석 표시:
  - 성적 적합도
  - 활동 적합도
  - 전형 적합도
- 개선 필요 영역 표시

**선행 Scenario:** WP13.3-1

---

## 📜 WP13.4 — AI 히스토리

### API 명세

| Method | Endpoint | 설명 |
|--------|----------|------|
| GET | `/api/ai/history` | AI 사용 히스토리 조회 |

---

### Scenario WP13.4-1: AI 히스토리 조회

**Given:**
- 이전에 AI 기능을 사용한 기록이 있다

**When:**
- AI 멘토 페이지에서 "히스토리" 탭 클릭

**Then:**
- 최근 AI 사용 기록 목록 표시
- 날짜, 유형, 요약 표시

**선행 Scenario:** WP13.1-1

---

### Scenario WP13.4-2: 히스토리 재활용

**Given:**
- AI 히스토리 목록이 표시되어 있다

**When:**
- 이전 분석 결과 클릭

**Then:**
- 해당 분석 결과 상세 표시
- "다시 분석" 버튼 제공

**선행 Scenario:** WP13.4-1

---

### Scenario WP13.4-3: 동일 질문 캐싱

**Given:**
- 30분 이내에 동일한 질문을 한 적이 있다

**When:**
- 동일한 질문으로 빠른 조언 요청

**Then:**
- 캐싱된 결과 즉시 반환
- "(캐시된 결과)" 라벨 표시

**선행 Scenario:** WP13.1-1
